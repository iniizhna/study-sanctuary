# Pomodoro Timer App 🍅

A Windows desktop Pomodoro timer app built with C# and WPF.

## Features
- ⏱️ 25-minute customizable timer
- 📝 Add notes while studying
- 😊 Track your mood
- 📊 Save sessions to local database
- 🎯 View study history

## How to Use

### Option 1: Download & Run (Easiest)
1. Go to **Releases** tab
2. Download the latest `.zip` file
3. Unzip and run `PomodorApp.exe`

### Option 2: Build from Source
1. Clone this repo
2. Open `PomodorApp.sln` in Visual Studio
3. Press F5 to run

## Installation
No installation needed! Just download and run the `.exe`

## Requirements
- Windows 10 or later
- .NET 6 runtime (included in the app)

## Tech Stack
- Language: C#
- Framework: WPF
- Database: SQLite
- IDE: Visual Studio
